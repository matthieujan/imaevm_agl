    • Issues with IMA/EVM

  • Not safe to operate
    • On iot hardware, if the system suffer a power outage, the hash may fail and brick the system
  • Not safe to protect
    • There is some ways (with sym and hard links) to bypass the integrity checks
  • Configuration hardware dependant
    • Without EVM, IMA is next to useless in security usage. However, the key installation requiring a safe way to store multiple and different keys (TPM), with harware solutions doesn't allow a generic solution in prod.
  • Experimental features
    • For the reasons above, and for maturity reasons, the features are explicit marked as deprecated in prod, and to use only in dev environment.

    Conclusion :

  • Not fit for embedded systems
  • Neither good security
  • Nor good safety
